Lights Out Game

Addendum

Process and Creative Choices

The Lights Out Game is a puzzle where the objective is to turn off all lights on a grid by clicking on them. Clicking a light toggles not only the clicked light but also its neighboring lights. This mechanic ensures that solving the puzzle requires careful strategy.

Game Logic: The board is initialized as a grid of lights, each of which can either be "on" or "off." The player toggles the lights by clicking them. A Hint button has been implemented, where after 10 clicks, the game is solved automatically, and all lights are turned off. The Refresh button resets the game to its starting state with a new random configuration.

Creative Choices: The board uses a grid layout to display the cells in a 5x5 grid. This grid is flexible and can be resized by adjusting the rows and cols constants. The CSS styles were chosen to create a simple and clean design. The game board is centered on the page with light gray colors for the "on" state and dark gray for the "off" state. Buttons are styled with a consistent color scheme and user-friendly spacing. The footer dynamically updates to show the last time the page was modified, ensuring that users are aware of when the content was last changed.

Challenges Faced: One challenge was ensuring that the Hint button works correctly and that the board responds properly to each click. Initially, there were issues where the lights were not toggling correctly when clicked, but this was resolved by ensuring that each light and its neighbors are toggled properly in the toggleLight function. Another challenge was making the hint mechanism clear and functional. The hint counter tracks how many times the user has clicked the Hint button. Once the counter reaches 10, all lights turn off, and a "You win!" message is displayed. This behavior was implemented using a simple counter and condition check.

Insights Gained: This project taught me how to use the DOM to dynamically create game elements like the game board and buttons. I also gained a deeper understanding of JavaScript event handling and state management within a game context.
