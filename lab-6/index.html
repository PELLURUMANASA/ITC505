Enter Numbers to Sort

Last updated:

Addendum:

This page was created as part of Lab 6 to implement a simple "Holy Grail" layout with a dynamic number sorting feature using JavaScript. The layout consists of a header, a sidebar menu, a main content area for interactive JavaScript functionality, an advertisement section with a clickable image leading to Amazon, and a footer that dynamically updates with the last modified date.

Challenges encountered during the implementation:

  • The first challenge was ensuring that the input values were properly validated before attempting to sort them. I had to make sure that only valid numbers would be processed, and that an appropriate error message was displayed if the user entered non-numeric data.
  • Another challenge was handling the dynamic updating of the last modified date in the footer. Initially, I had difficulty ensuring the date updated correctly, but this was resolved by using JavaScript to retrieve the document's last modified timestamp and displaying it in the footer.
  • Additionally, ensuring that the "Refresh" button cleared both the input field and the output area required some attention to make sure the page behaved as expected without reloading.
